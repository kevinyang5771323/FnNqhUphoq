# 项目简介

## 核心功能
- 系统设置
- 用户注册和登录
- 宠物食品浏览
- 宠物食品购买
- 宠物食品订单管理
- 在线支付
- 物流跟踪

## 适用场景
- 宠物食品销售商家
- 宠物食品消费者

## 技术特点
- 后端：采用 Django 框架，实现前后端分离
- 前端：使用 Vue.js + ElementUI 框架，提供友好的用户界面
- 开发环境：支持 Python3.9 以上，Pycharm 编辑器，MySQL5.7/MySQL8 数据库，VSCode 代码编辑器
